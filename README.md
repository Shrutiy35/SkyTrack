This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).



 Build a complete modern frontend web app called "SkyTrack ✈️- Live Flight Tracker" using Google's  Anti-Gravity IDE.    
 
  # Goal:
  
A user enters a flight number (e.g., AI302, EK501) , and the app displays live flight status using any free flight API or mock JSON data.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
Navigate to the project directory: cd skytrack

Run the command:

npx next dev -p 3001
